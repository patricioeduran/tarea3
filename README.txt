Este es un repositorio para la Tarea 3 del curso FIS1220 - Programación, específicamente para el apartado 3.a. 

# Información relevante

Inicializar un repositorio de Git en una carpeta es conveniente porque permite llevar un control de versiones, lo que facilita el seguimiento de cambios y la recuperación de versiones anteriores en caso de errores. También mejora la colaboración en equipo, ya que varios usuarios pueden trabajar simultáneamente en distintas partes del mundo de forma remota.

Además, tener un repositorio organizado para cada proyecto facilita la gestión y separación de tareas. Por último, permite respaldar el trabajo en la nube utilizando plataformas como GitHub, asegurando que no se pierda información importante.
